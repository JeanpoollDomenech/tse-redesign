# TSE Redesign
Este proyecto consiste en un rediseño de la interfaz para el sitio web gubernamental Tribunal Supremo de Elecciones usando los conocimientos de usabilidad, accesibilidad, UI, UX.

Se abarcara de manera completa la interfaz inicial (posible reestructuración) y las consultas civiles, se debe abarcar un 40-50% del sitio web.

## Requisitos

- Node.js (versión 18 o superior)
- npm (incluido con Node.js)
- Git (opcional, para control de versiones)

Verificar versiones:

```bash
node -v
npm -v
```

---

# Instalación y Ejecución

## 1. Clonar el repositorio

```bash
git clone https://github.com/JeanpoollDomenech/tse-redesign
```

---

## 2. Entrar al directorio del proyecto

```bash
cd tse-redesign
```

---

## 3. Instalar dependencias

```bash
npm install
```

---

## 4. Ejecutar el proyecto

```bash
npm run dev
```

El proyecto estará disponible en:

```txt
http://localhost:5173
```

# Toma de Decisiones

## ¿Por qué React?

Se eligió React debido a:
- Su arquitectura basada en componentes
- Facilidad para reutilizar código
- Amplia comunidad y documentación
- Escalabilidad para proyectos medianos y grandes

---

## ¿Por qué Vite?

Se decidió usar Vite porque:
- Tiene tiempos de inicio muy rápidos
- Mejor experiencia de desarrollo
- Configuración sencilla
- Excelente integración con React

---

## ¿Por qué TailwindCSS?

TailwindCSS fue seleccionado por:
- Rapidez al desarrollar interfaces
- Facilidad para crear diseños responsive
- Consistencia visual
- Menor necesidad de escribir CSS personalizado

---

## ¿Por qué React Router DOM?

Se implementó React Router DOM para:
- Manejar múltiples vistas dentro de la aplicación
- Mejor experiencia de navegación
- Organización más clara de páginas y componentes
